ESTILO

CSS3

A maquiagem. O CSS transforma texto em arte. Com Flexbox e Grid, dominamos o espaço bidimensional.

Fórmula Pro: box-sizing: border-box; evita que o padding aumente o tamanho real do elemento!

.hero {
  display: grid;
  place-items: center;
  transition: all 0.5s ease;
}
